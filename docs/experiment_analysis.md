## 2. 핵심 발견

### 2.1 Skip Filter (σ=1.0)는 **두 데이터셋 모두에서** 일관되게 대폭 개선

| 데이터셋 | Baseline | +Skip Filter | +Hybrid Ellipse | 총 향상률 |
|---------|----------|-------------|-----------------|---------|
| Swirski (Total) | 0.5831 | 0.6937 | **0.7812** | **+33.9%** |
| LPW (1.avi) | 0.4987 | 0.7355 | **0.8291** | **+66.2%** |

**Cross-dataset 일반화 성공:** Swirski(가시광, 정지영상)와 LPW(가시광, 비디오)라는 서로 다른 unseen 데이터셋에서 동일한 방법이 동일한 방향으로 대폭 개선됨. 이것은 Skip Connection의 고주파 오염이 특정 데이터셋에 국한된 현상이 아니라 **구조적(architectural) 문제**임을 입증합니다.

### 2.2 σ=1.0이 최적점, σ↑는 성능 단조 감소

```
Swirski mIoU vs σ:
σ=0.0  ████████████████████████████████████████████ 0.5831
σ=1.0  ████████████████████████████████████████████████████████ 0.6937 ← 최적
σ=1.5  ██████████████████████████████████████████████████████ 0.6681
σ=2.0  ████████████████████████████████████████████████ 0.6057
σ=2.5  ██████████████████████████████████████ 0.4964
σ=3.0  ██████████████████████████████ 0.3716
```

**σ=1.0을 넘기면 성능이 단조 감소**합니다. 이는 σ가 커질수록 동공 경계 정보까지 파괴되기 때문입니다. "더 강한 필터 = 더 나은 성능"이 아님을 보여줍니다.

**연구자님의 직관 ("필터 강도를 높여야 한다")에 대한 객관적 평가:**
- σ=1.0에서 여전히 속눈썹 분절이 남아있는 프레임이 존재하는 것은 사실
- 그러나 σ를 올리면 **이미 성공하고 있는 프레임들의 성능이 더 빠르게 하락**
- net effect는 σ>1.0에서 항상 음수
- 속눈썹 분절 잔존 프레임은 Skip filter가 아닌 **다른 방법(Ellipse 개선 등)**으로 보완해야 함

### 2.3 RITnet 전처리 (Gamma 0.8 + CLAHE)는 해가 됨

| 조건 | Swirski | LPW |
|------|---------|-----|
| Baseline | 0.5831 | 0.6319 |
| + RITnet Preprocess | 0.5761 (**−0.0070**) | 0.6075 (**−0.0244**) |
| Filter σ=1.0 | 0.6937 | 0.7797 |
| Filter σ=1.0 + Preprocess | 0.6685 (**−0.0252**) | 0.7467 (**−0.0330**) |

**4가지 실험 모두에서 전처리가 성능을 하락시켰습니다.**

**객관적 원인 분석:**

1. **Domain mismatch 심화:** TransUNet은 OpenEDS(근적외선) 이미지를 raw 상태로 학습했습니다. Gamma 보정과 CLAHE는 이미지의 밝기/대비 분포를 변형시켜, 모델이 학습한 feature 분포와 더 멀어지게 만듭니다.

2. **RITnet과 TransUNet의 학습 조건 차이:** RITnet은 이 전처리를 **학습 시에도 적용**했기 때문에 효과가 있었습니다. TransUNet에 추론 시에만 적용하면, 학습 분포에서 벗어나는 역효과.

3. **CLAHE의 부작용:** 국소 대비 향상은 속눈썹 에지를 **더 선명하게** 만들어 오히려 분절을 악화시킬 수 있습니다.

> **결론: RITnet 전처리를 TransUNet에 이식하는 것은 비권장. 전처리로 밝기를 정규화하려면 TransUNet을 해당 전처리를 적용한 데이터로 재학습해야 합니다.**

### 2.4 p1-right(과노출)는 어떤 방법으로도 해결 불가

| 방법 | p1-right mIoU |
|------|-------------|
| Baseline | 0.3504 |
| Skip Filter σ=1.0 | 0.3506 (+0.0002) |
| Filter + Preprocess | 0.3592 (+0.0088) |
| SAM Refine | 0.4960 |

과노출은 **동공 영역의 픽셀 값 자체가 배경과 동일**한 문제이므로:
- Skip의 고주파를 제거해도 효과 없음 (동공 signal 자체가 부재)
- 입력 전처리로 밝기를 낮춰도 동공 내부 텍스처가 복원되지 않음
- 유일한 해법은 입력 차원에서 동공을 복원하거나(불가능), 과노출에 robust한 학습이 필요

### 2.5 글린트(하얀 반사점) 제거 가능성

Skip Filter의 Gaussian blur는 **글린트도 감쇠시킬 수 있습니다.** 글린트는 1~3px 크기의 점(point)으로 주파수적으로는 최고 대역입니다. σ=1.0의 Gaussian blur는 이를 주변 픽셀과 평균화하여 자연스럽게 억제합니다.

LPW 오버레이를 보면 baseline에서 글린트 주변에 소량의 spurious prediction이 있었지만, Skip Filter 적용 후 대부분 제거되었습니다.

다만, 글린트가 동공 **중심부**에 있으면 blur로도 동공 내부 feature를 교란할 수 있습니다. 이 경우는 Ellipse fitting으로 보완해야 합니다.

## 3. Ellipse 후처리의 정밀 분석 및 실패

### 3.1 Largest Blob 피팅의 위양성 문제

Exp C(0.6937) > Exp D Largest(0.6896) — **가장 큰 blob만 피팅할 때의 한계:**
속눈썹에 의해 동공이 2개로 분절되면, 절반짜리 조각에 타원을 맞추므로 위치/크기가 크게 어긋나며 p1-left에서 위양성이 심하게 발생했습니다.

### 3.2 '합산 피팅 (All-blob)'의 치명적 부작용

이 문제를 해결하기 위해 **모든 contour의 점을 합산(`np.vstack(contours)`)한 뒤 피팅**하는 방식을 Swirski와 LPW에 적용(Exp D2)했습니다. 결과는 예상과 달랐습니다.

- **Swirski Total:** 0.6937 → 0.6518 (하락)
  - *p1 케이스:* 0.7117 → 0.7500 개선 (속눈썹 분절 극복 성공)
  - *p2 케이스:* 0.8822 → 0.6666 **대폭락**
- **LPW Folder-1:** 0.7797 → 0.6765 **대폭락**

**원인 분석:**
동공 외곽에 아주 작은 **글린트(Glint) 노이즈**나 속눈썹 노이즈가 1~2px이라도 독립된 blob으로 남게 될 경우, 합산 피팅 로직은 **동공과 저 멀리 떨어진 노이즈 점을 모두 감싸는 거대한 타원**을 그려버립니다. 이로 인해 거대한 위양성이 발생하며, 깨끗했던 p2 케이스와 LPW의 성능이 붕괴되었습니다.

### 3.3 해결: 하이브리드(형태학+임계값) 타원 피팅 (최종 모델)

단순히 Largest blob을 고르거나 All blob을 합치는 것의 문제를 해결하기 위해, 연구자님의 직관을 반영한 **하이브리드 후처리 파이프라인**을 구현했습니다:

1. **Morphological Closing:** 9x9 Ellipse 커널로 닫기 연산을 수행하여 속눈썹 두께 수준의 얇은 틈새를 기하학적으로 우선 접합.
2. **Thresholding (크기/거리 필터링):**
   - 크기: 가장 큰 blob 대비 5% 미만인 조각은 노이즈로 간주하여 버림.
   - 거리: 가장 큰 blob 중심점 대비 반경 50px을 초과하는 조각은 노이즈로 간주하여 버림.
3. 이 필터를 통과한 **유효한 조각들만** 합산하여 타원 피팅.

**결과: 초대박 성공**
이 로직은 글린트 등 멀리 떨어진 노이즈를 완벽하게 배제하면서도 쪼개진 동공 덩어리만 안전하게 합쳐주었습니다.
- **Swirski Total mIoU:** 0.6937 → **0.7812** (SAM 모델인 0.7278을 **능가**함!)
- **LPW 1.avi:** 0.7355 → **0.8291** (초정밀 타겟팅)

> **결론:** 연구자님의 직관대로 거리/크기 임계값을 설정하여 노이즈를 제외한 뒤 피팅하는 방식이 완벽하게 적중했습니다. TransUNet의 예측력이 SAM ViT-H를 넘어서는 결과를 얻어냈습니다.

## 4. 향후 개선 방향 (재평가)

| 순위 | 개선 | 근거 | 기대 효과 | 리스크 |
|------|------|------|---------|------|
| **1** | **Morphological Closing (형태학적 닫기)** | 합산 Ellipse가 노이즈에 취약함이 입증됨. 단순히 좁은 틈(속눈썹)만 메워 위양성을 최소화. | Ellipse 대안 | 낮음 |
| **2** | **Skip별 독립 σ (σ₀, σ₁ 분리)** | probe에서 Skip#0, #1의 HF 격차가 다름 | 미세 최적화 | 낮음 |
| 3 | 면적 기반 Ellipse 노이즈 제거 | 합산 전, 전체 면적의 10% 미만인 소형 blob을 삭제 후 피팅 | 노이즈 저항성 | 중간 |

> **비권장: σ 증가 (1.5 이상)** — σ sweep 결과가 단조감소를 명확히 보여줌. 속눈썹 분절 잔존 케이스는 σ가 아닌 Ellipse 개선으로 해결해야 함.

> **비권장: RITnet 전처리** — 4가지 실험 전부에서 하락. TransUNet과 domain mismatch.
